Лабораторная работа №2
		
Продемонстрировать работу с разными типами коллекций: 
	1. массивы
	2. класс нетипизированных элементов ArrayList
	3. класс типизированных элементов List<>

Выполнить любое задание на выбор с каждым из типов коллекций:

	1. Имея набор слов/чисел отсортировать любым алгоритмом (реализация алгоритма самостоятельная, не использовать встроенную сортировку)
	2. Имея набор слов/чисел сформировать список повторяющихся
	3. Имея набор слов подсчитать количество гласных/согласных букв
	4. Имея набор чисел определить максимальный (самый широкий) диапазон чисел, идуших подряд
	5. Имея набор чисел определить минимальное и максимальное значение
	6. Имея набор слов определить количество слов с английскими буквами и без английских букв
